# Medical Visit Manager

Aplikacja konsolowa do zarządzania pacjentami i wizytami lekarskimi.
---

## 🛠 Technologie

- Java 8  
- Czysty JDBC  
- Baza danych: Firebird
---

## 📌 Funkcjonalności

- Dodawanie pacjentów  
- Rejestracja wizyt z powiązaniem do pacjenta  
- Wyświetlanie wizyt wybranego pacjenta  
- Obliczanie sumy kwot wizyt w zadanym okresie  
- Rejestracja pacjenta i wizyty w jednej transakcji
